Script Validator

This utility allows you to quickly indent your scripts while simultaneously checking it for several errors, many of which are not picked up by the GECK's compiler. To use the validator, simply paste or type the code that you wish to validate into the top text box on this page.

If you find a bug, please report it so I can fix it.
